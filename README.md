Script used for decompress files in a generic way.

<b>Usage:</b>
./exe.sh file

<b>Suported formats:</b>
- .tar.bz2
- .tar.gz
- .bz2
- .rar
- .gz
- .tar
- .tbz2
- .tgz
- .zip
- .Z
- .7z

<b>Recomended:</b>
- Put it inside a function in your file <i>~/.bash_aliases</i>
- This way you can call it anywhere
